# First Login Guide

## 1. Access the Application

**Web:** Open http://localhost:3000 (development) or your Vercel URL (production)  
**APK:** Install the downloaded APK on your Android device

## 2. Login with Test Account

Use the seeded admin account:
- **Email:** `admin@cyberfrost.vercel.app`
- **Password:** `Cyber123!`

After first login, you'll land on the **Executive Dashboard**.

## 3. Enable 2FA (Recommended)

1. Go to **Settings → Profile**
2. Click **"Enable 2FA"** under Security
3. Scan the QR code with Google Authenticator or Microsoft Authenticator
4. Enter the 6-digit code to verify
5. 2FA is now active

## 4. Explore the Dashboard

- **Executive View** — Real-time security posture
- **Attack Surface** — Discovered domains and assets
- **Vulnerabilities** — CVE database with search
- **Threat Intelligence** — MITRE ATT&CK framework
- **Action & Mitigation** — Takedown requests and blocking
- **Integrations** — Connect SIEM, Jira, ServiceNow

## 5. Create Your First Scan

1. Navigate to **Attack Surface**
2. Click **"Start Scan"**
3. Enter your domain
4. Monitor progress in real-time

## 6. Next Steps

- Add team members (Admin → Users)
- Configure integrations (Integrations page)
- Set up notification preferences
- Explore AI-powered threat analysis
